# CBB Catalog Template

## Purpose

This catalog tracks Common Building Blocks (CBBs) — reusable components, libraries, and patterns that should be preferred over bespoke solutions per Constitution Principle IV:

> "CBB (Common Building Block) reuse MUST be preferred over bespoke solutions"

## Instructions

- **Adding entries**: When a new reusable component is created, add it to the table below.
- **Using existing CBBs**: Before creating a new component, check this catalog first. If a suitable CBB exists, reuse it.
- **Reviewing**: Each CBB should be reviewed at least once per release cycle.

## New Component Checklist

When creating a NEW component (not reusing an existing CBB):
- [ ] Has the CBB catalog been searched for an existing reusable alternative?
- [ ] If no CBB found, is the new component designed for future reuse?
