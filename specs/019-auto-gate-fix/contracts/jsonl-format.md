# 契约：JSONL 门控记录格式

**版本**: 1.0
**用途**: 定义 gate-history.jsonl 的写入和读取契约

## 写入方

| 脚本 | 写入时机 | 写入内容 |
|------|----------|----------|
| gate-record.ps1 | 每次门控记录操作 | 完整的 GateRecord JSON 行 |

## 读取方

| 组件 | 读取时机 | 用途 |
|------|----------|------|
| generate-report.ps1 | 生成 HTML 报告时 | 读取所有记录渲染报告 |
| vipd-report SKILL.md | 用户查看报告时 | 展示门控历史 |

## 格式规范

- 每行一个完整的 JSON 对象，以 `\n` 结尾
- 不允许在一行内出现换行符
- 文件编码: UTF-8 without BOM
- 追加写入，不修改已有行
